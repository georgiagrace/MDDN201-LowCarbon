Dithered Images

Dithering images is a form of optimising images into smaller file sizes. Dithering images reduces the colour information in an image, thereby decreasing the file size hugely. Luckily they maintain a balance between visual appeal and data load (aka reducing the energy required to transmit and render the webpage)– happy file loading!

Default Typeface

By using a default typeface, we remove the need for the browser to fetch and load external font files. Yes, they might look cool – but so is the environment. Default typeface ensures faster page load speed which is a crucial factor for simplifying the user experience. PLUS a faster website is more energy-efficient. Default typeface = Less processing power!

No Third-Party Tracking,
No Advertising Services, No Cookies.

Eliminating third-party tracking helps protect user privacy and reduces unnecessary data transfers, thereby making a website more energy-efficient and user-friendly. Also, avoiding the use of advertising services, a website can drastically reduce its energy consumption per page load. COOKIES! We love to eat them but these small data files contribute to large data transfer loads and are not energy-efficient.

Importance of Low Carbon Web Design to Designers

As future designers, adopting these low-carbon web design approaches is a progressive step towards reducing the digital carbon footprint. They will help align with our personal and professional sustainability goals. Educate your co-workers, friends and peers to ensure we can ensure a greener, more sustainable tomorrow.
